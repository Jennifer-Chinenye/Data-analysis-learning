# Business Sales Data Cleaning and Analysis.
## Overview
This Project is a practice in analyzing a Business dataset using Excel. The dataset contains:
-Customers Fullname
-Order Number
-Quantity Ordered
-Unit Price
-Order Date
-Order Status
-Product Line
-Quarter ID
-Year ID
-Deal Size
-Companies
## Skills Demonstrated
-Data Cleaning and Formatting
-Removing Duplicates and Correcting errors
-Sorting and Filtering
-Used Pivot tables to answer questions
-Created Simple Dashboard by Combining Pivot Chart in one sheet.
## Insights from the Pivot Tables
-Total Revenue by Product Line
-Top 10 Companies by Total Revenue
-Revenue by Deal Size
-Average Sales by Month
-Number of Transactions by Year
-Customer's Status
## Files
-**Business_sales Analysis.XLSX**- File containing all sheets (Raw, Cleaned, and Analyzed).
## Summary
This Project demonstrates my ability to clean raw business data, analyze it with Pivot Tables, and present clear insights using Excel.
